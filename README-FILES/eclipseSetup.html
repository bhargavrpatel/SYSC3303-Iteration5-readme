  1. Open Eclipse; Select File > Import

    2. Click To Enlarge
  2. From the Dialogue box, click "Existing Projects into workspace" and then click Next

    3. Click To Enlarge
  3. Using the Browse.. option, select our project folder and click Finish.

    4. Click To Enlarge
  4. Your Package Explorer should now look like this. To setup the Test files, run the setupTestFiles.java file. You will see command prompt pop up and close very quickly, this is the hiddenFiles.bat file in the root of our project running. It made two files, muffins.txt (client-files) and chocolate.txt (server-files) hidden. You will also see the Console output saying that two other files, cantEatThis.txt (client-files) and bananaCake.txt (server-files) have been made of format readonly.

    5. Click To Enlarge

    Click To Enlarge
    After running setupTestFiles.java, you will see that some files are now hidden and readonly. Please goto Files\client-files folder and ensure muffins.txt is hidden and cantEatthis.txt is readonly. Some University computers block Batch files from executing. We are using Batch files to make the muffins.txt in client and chocolate.txt in server hidden.
