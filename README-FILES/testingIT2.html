  3. You can test any of the following cases using their respective steps. It is important to restart errorSim.java each time for clarity and to reset the settings, and also recommended you delete the files from the target folder.
    1. Duplicate Packets
      1. in the errorSim.java console, type "10" and press enter
      2. You will be asked to specify whether you wish to duplicate ACK or DATA packet. Select either by typing 'ack' or 'data' and press enter.
      3. You will now be asked to specify the block you wish to duplicate, type an integer in and press enter.

    2. Testing Delayed Packets
      1. in the Error Simulator console, type '9' and press enter to select Delaying packets
      2. You will be asked to specify whether you wish to duplicate ACK or DATA packet. Select either by typing 'ack' or 'data' and press enter.
      3. You will now be asked to specify the block you wish to duplicate, type an integer in and press enter. Note: I have ommited an image here to reduce submission size, simply refer to the example image above. Everything is identical except in this case, you entered '9'.
      4. Skip to Step #4 and run the Client.java file
    3. Testing Lost Packets
      1. in the Error Simulator console, type '8' and press enter to select Lose packets
      2. You will be asked to specify whether you wish to duplicate ACK, DATA, or REQUEST packet. Select either by typing 'ack' or 'data' or 'request' and press enter.
      3. IF you selected Data or ACK, You will now be asked to specify the block you wish to duplicate, type an integer in and press enter. Note: I have ommited an image here to reduce submission size, simply refer to the example image above. Everything is identical except in this case, you entered '9'.
