Below is a visualization of all the files and folders, and their brief descriptions in this submission
  1. .classpath Created by Eclipse
  2. .project Created by Eclipse
  3. README.html The main Index of README htmls
  4. hideFiles.bat A bat file used by setupTestFiles.java to make certain test files hidden. Do not run this yourself.
  5. Diagram
    • Class Diagrams

      Contains the class diagrams

    • UCM Diagrams Contains the use case map diagrams
    • Timing Diagrams Contains the sequence/timing diagrams per each case
  6. Files
    • client-files Contains the test files associated with the Client. Files sent from server will also be here.
      • loremClient.txt A normal file
      • muffins.txt After running setupTestFiles.jar, this will be hidden
      • cantEatThis.txtAfter running setupTestFiles.jar, this will be readonly
      • 512.txtA file that is exactly 512 bytes
      • largerthan132kb.txtA file that is 150 kb
    • server-files Contains the test files associated with the Server. Files sent from client will also be here.
      • loremServer.txt A normal file
      • chocolate.txt After running setupTestFiles.jar, this will be hidden
      • bananaCake.txtAfter running setupTestFiles.jar, this will be readonly
      • 1536.txtA file that is exactly 512*3 bytes (A multiple of 512)
  7. src
    • Client.jar Main Client class
    • errorSim.jar Main Error Simulator class
    • Server.jar Main Server class
    • reqListener.jar The listener class, used when listening to requests
    • serverTransfer.jar A class extending Thread, used in transfer by server
    • common Folder containing class used by more than one classes
      • setupTestFiles.java Run this class BEFORE ANY! it makes test files hidden/readonly
      • abortException.java A custom exception class that we throw
      • ConnectionManager.java A major class we use to manage connections, requests, and sending/recieveing
      • packets A folder for the Packet class and its subclasses for different types of packets.
        • Packets.java An abstract class used by all the other packet type classes
        • ACKPackets.java
        • DataPackets.java
        • RequestsPackets.java
        • DataPackets.java
